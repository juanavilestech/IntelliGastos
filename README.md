# Gastos App

Aplicación web para el registro y análisis de gastos personales.

## Objetivo
Permitir al usuario:
- Registrar gastos diarios
- Visualizar y filtrar gastos
- Obtener análisis automáticos mediante IA

## Stack tecnológico
- Frontend: React
- Backend: Node.js + Express
- Base de datos: PostgreSQL
- IA: Python (microservicio independiente)

## Arquitectura
- Monorepo simple
- Arquitectura en capas
- Microservicio para IA

## Motivación
Proyecto orientado al aprendizaje práctico de:
- Bases de datos relacionales
- Arquitectura backend
- Integración de IA en sistemas reales
